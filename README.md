## üßæ Overview

**Fluent Aurora** is a lightweight, open-source **audio player** for Windows.  
It brings a modern, responsive Fluent UI to your music experience, with fast indexing and intuitive playlist management ‚Äî all powered by the .NET ecosystem.

---

## ‚ú® Features

- Built with modern **.NET** and **Avalonia (FluentAvalonia) UI** stack
- Lightweight, open-source, and blazing fast
- Smart **music library indexer** with SQLite backend
- **Custom playlists** with tag-based organization
- Sleek Fluent UI experience consistent with Windows design
